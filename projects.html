StockScope Pro

Tech Stack: Python, Streamlit, Plotly, PRAW, NewsAPI, SEC API, VADER NLP

Real-time investment research dashboard that combines social media, news, and SEC filings to generate AI-powered BUY/HOLD/SELL recommendations with interactive risk breakdowns.

  • Built with Streamlit & Plotly on a Python backend
  • Scrapes Reddit, financial news, and SEC filings
  • Runs VADER NLP to calculate sentiment and confidence
  • Supports auto-refresh, CSV export, and responsive layout

This project reflects my passion for turning noisy data into visual, actionable insights — and taught me how to scale real-time dashboards and optimize UX for serious analysis.
