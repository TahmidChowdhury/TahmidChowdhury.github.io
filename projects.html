Victaulic Platform Refactor

Tech Stack: Angular 11, NestJS, TypeORM, MySQL, SQL Server, GraphQL, Azure Service Bus, Docker, Autodesk Forge API, MuleSoft, Mendix

Led the comprehensive modernization of Victaulic's legacy enterprise platform used by global manufacturing teams, transforming monolithic architecture into a scalable microservices ecosystem serving thousands of users worldwide.

Key Achievements:

  • Microservices Architecture: Decomposed monolithic application into 15+ modular NestJS services including project management, material planning, fabrication tracking, and document generation
  • Database Optimization: Implemented TypeORM with MySQL/SQL Server integration, created 50+ database migrations, and optimized complex queries for enterprise-scale data processing
  • API Gateway & GraphQL: Built unified GraphQL API layer serving Angular frontend with 100+ resolvers and REST controllers for external integrations
  • Enterprise Integrations: Developed seamless connections with Autodesk Forge (BIM360), MuleSoft ESB, Mendix low-code platform, and Azure Service Bus for real-time data synchronization
  • Performance Engineering: Reduced application load times by 60% through code splitting, lazy loading, and efficient caching strategies
  • CI/CD Pipeline: Architected Azure DevOps workflows with automated testing, Docker containerization, and multi-environment deployments
  • Module Federation: Implemented Nx monorepo structure with shared libraries, enabling independent team development and deployment cycles

Technical Leadership:

  • Mentored 2+ junior developers through architectural reviews and pair programming sessions
  • Established coding standards, testing protocols, and documentation practices across engineering teams
  • Reduced deployment time from 4 hours to 30 minutes through infrastructure automation
  • Improved system reliability with comprehensive error handling, logging, and monitoring solutions

This project showcases my expertise in large-scale enterprise architecture, leading cross-functional teams, and delivering mission-critical software that directly impacts manufacturing operations across multiple continents.
