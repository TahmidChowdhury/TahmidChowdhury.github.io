BankStack — Personal Finance Platform

Tech Stack: NestJS, TypeScript, PostgreSQL, Prisma, Plaid

BankStack is a personal finance platform that integrates with Plaid to track balances, transactions, and debt payoff strategies. It focuses on consolidating financial data, modeling repayment workflows, and providing actionable insights through backend-driven logic.

  • Built backend services using NestJS, TypeScript, and PostgreSQL with Prisma
  • Integrated Plaid API for account linking, balance sync, and transaction ingestion
  • Designed relational data models for accounts, transactions, and financial workflows
  • Implemented backend logic for debt payoff strategies and financial planning

StockScope Pro — Market Sentiment Platform

Tech Stack: Next.js, Python, FastAPI, Docker, VADER

StockScope Pro is a data-driven platform that ingests and processes financial news and social sentiment through API pipelines and background workflows. It focuses on efficient ticker-based analysis, caching strategies, and transforming unstructured data into structured sentiment signals.

  • Built a full-stack platform with Next.js and FastAPI for data ingestion and analysis
  • Designed API pipelines to aggregate financial news and Reddit sentiment
  • Implemented background processing and caching to reduce redundant requests
  • Developed sentiment analysis workflows using VADER

Enterprise Integration Platform — Victaulic

Tech Stack: Angular, NestJS, TypeORM, MySQL, Azure Service Bus, Autodesk Forge, MuleSoft, Docker

An enterprise integration platform supporting data synchronization across internal and external systems in manufacturing and construction workflows.

  • Modernized monorepo into separated frontend/backend services to reduce coupling and improve deployment flexibility
  • Upgraded stack from Node.js 12 to 22, Angular 11 to 16, and NestJS 7 to 10+
  • Built backend services using NestJS and Azure Service Bus for event-driven workflows
  • Enabled near real-time data synchronization across internal and external systems
  • Supported integrations with Autodesk Forge and MuleSoft
  • Improved system reliability by resolving production issues and refactoring legacy code
